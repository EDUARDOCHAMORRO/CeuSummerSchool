INTRODUCCIÓN A LA VISUALIZACIÓN DE DATOS

Al final de este curso los alumnos habrán obtenido las nociones básicas para empezar a trabajar con bases de datos abiertas; utilizar Qgis para importar datos, mapearlos, exportar e imprimir; Excel para importar, limpiar y exportar bases de datos, trabajar con tablas dinámicas; Rawgraphs para importar bases de datos y trasladarlos a gráficos; Illustrator como herramienta final de retoque y maquetación.

Fuentes de datos

Las bases de datos a emplear serán las encontradas en el catálogo de datos abiertos de Madrid a escala ciudad (https://datos.madrid.es/portal/site/egob/menuitem.9e1e2f6404558187cf35cf3584f1a5a0/?vgnextoid=374512b9ace9f310VgnVCM100000171f5a0aRCRD&vgnextchannel=374512b9ace9f310VgnVCM100000171f5a0aRCRD&vgnextfmt=default) y del país de Perú a escala país (http://www.datosabiertos.gob.pe/)
